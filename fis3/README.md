# fis3

> some fis3 project

使用官方构建工具vue-cli搭建，主要的依赖库：

## Configuration
> 默认为fis.conf.js文件
```
npm install -g fis3
```
## Build Setup

``` bash
# 构建发布到当前项目下的 output 目录下
fis3 release -d ./output

# 本地发布
fis3 release

# 启动本地 Web Server http://127.0.0.1:8080
fis3 server start

```
